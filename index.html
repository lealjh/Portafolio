Web Designer

Proceso de crear la apariencia visual y la interfaz de usuario de un sitio web. Implica elementos como el diseño, colores, tipografía, imágenes, interacción y accesibilidad.

Frontend Developer

Profesional que se ocupa de la parte visible y funcional de sitios web y aplicaciones. Su enfoque está en la interfaz de usuario y experiencia del usuario. Utiliza HTML, CSS y JavaScript para diseñar y desarrollar la apariencia y la interacción.

Backend Developer

Un desarrollador backend maneja la lógica, funcionalidad y seguridad del servidor de un sitio web o aplicación. Colabora con los desarrolladores frontend para brindar una experiencia completa al usuario.

Sobre Mi

Biografia

Soy Daniel Alexander Pérez Leal, un apasionado estudiante de 18 años de tecnología en análisis y desarrollo de software. Me especializo en desarrollo frontend y disfruto diseñando interfaces atractivas y funcionales. Aprendo de cada proyecto y espero seguir creciendo y contribuyendo con soluciones innovadoras en el futuro.

Experiencia

Trabajé en diversos proyectos como desarrollador frontend, destacando mi experiencia en un equipo de desarrollo para un sitio web de comercio electrónico. Además, he trabajado con bases de datos MySQL y aprendido la importancia del trabajo en equipo y la colaboración efectiva. Estas experiencias han fortalecido mi pasión por el desarrollo frontend y me han dado confianza para enfrentar nuevos desafíos en el futuro. Estoy emocionado de seguir creciendo en este campo y crear soluciones impactantes en el mundo digital.
